CSS Length Units

There are two types of length used in css

  • Absolute length units
  • Relative length units

Absolute length units

They are not relative to anything else, and are generally considered to always be the same size.

  1. cm -> Centimeters -> 1cm = 37.8px
  2. mm -> Millimeters -> 1mm = 1/10th of cm
  3. Q -> Quarter-millimeters -> 1Q = 1/40th of 1cm
  4. in -> Inches -> 1in = 2.54cm = 96px
  5. pc -> Picas -> 1pc = 1/6th of 1in
  6. pt -> Points -> 1pt = 1/72nd of 1in
  7. px -> Pixels -> 1px = 1/96th of 1in

Relative length units

Relative length units are relative to something else, for example font size of parent, or size of viewport.

  1. em -> font size of the parent.
  2. rem -> font size of the root element.
  3. vw -> 1% of the vieport's width
  4. vh -> 1% of the viewport's height
  5. vmin -> 1% of the vieport's smaller dimension
  6. vmax -> 1% of the viewport's larger dimension
I am 20px wide
I am 20vw wide
I am 20em wide
